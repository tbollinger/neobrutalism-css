A softened-neobrutalism CSS pack — bold type, hard offset shadows, a satisfying press effect on every interactive surface. Framework-agnostic, no build step to consume, and the whole look reskins from a handful of custom properties.

Install

Pick one. The CSS works on its own; the JS helper is optional (it auto-wires modals & dropdowns).

CDN — jsDelivr serves the repo directly. Drop these in your <head> / before </body>. 
<link rel="stylesheet"
      href="https://cdn.jsdelivr.net/gh/tbollinger/neobrutalism-css@v1.0.0/dist/neobrutalism.min.css">
<script src="https://cdn.jsdelivr.net/gh/tbollinger/neobrutalism-css@v1.0.0/dist/neobrutalism.min.js"></script>
npm — install straight from GitHub (no registry publish needed), then import. 
npm install github:tbollinger/neobrutalism-css#v1.0.0

// then, in your entry file:
import 'neobrutalism-css/css';   // styles
import 'neobrutalism-css/js';    // optional behavior helper
Or just copy the files — grab dist/neobrutalism.css (+ .js) and link them yourself. Zero dependencies, zero build. 
<link rel="stylesheet" href="neobrutalism.css">
<script src="neobrutalism.js"></script>

Tags and badges. Set --nb-pill-color inline to fill; add .nb-pill for fully-rounded; --interactive for clickable.

Modal

Dialog wired by the JS helper: data-nb-open="#id" opens, data-nb-close / backdrop click / Esc closes.

Hello there

Close with the ×, the Esc key, or by clicking the backdrop.

<button class="nb-btn nb-btn--primary" data-nb-open="#demo-modal">Open modal</button>

<div class="nb-modal" id="demo-modal" hidden>
  <div class="nb-modal__panel">
    <button class="nb-modal__close" data-nb-close aria-label="Close">×</button>
    <h2>Hello there</h2>
    <p>Close with the ×, Esc, or the backdrop.</p>
    <button class="nb-btn" data-nb-close>Got it</button>
  </div>
</div>

JS helper

Include neobrutalism.js and it auto-wires on DOMContentLoaded — no config. The CSS works fully without it; the helper only adds modal & dropdown behavior.

  • Modals — [data-nb-open="#id"] opens; [data-nb-close], backdrop click, or Esc closes.
  • Dropdowns — [data-nb-toggle] toggles; outside-click or Esc closes.
  • Inject markup later? Call Neobrutalism.init() again — it's idempotent.
// auto-wires itself on load — usually nothing to do.

// after injecting new markup (idempotent):
Neobrutalism.init();

// drive things manually if you skip the data-attributes:
Neobrutalism.open(el);
Neobrutalism.close(el);

Theming

Every value is a CSS custom property. Override in your own :root for a global reskin, or on any scoped container for a local one — no recompile.

Drop this anywhere after the pack's stylesheet: 
:root {
  --nb-border-color: #1a1a2e;
  --nb-radius: 0;          /* hard-edge variant */
  --nb-accent: #ff5470;
  --nb-shadow-size: 6px;
}
